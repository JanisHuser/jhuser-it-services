# J.Huser IT-Services

Website for [jhuser-it-services.ch](https://jhuser-it-services.ch) – built with Next.js and exported as a static site.

## Local Preview

### 1. Install dependencies

```bash
npm install
```

### 2. Start the dev server (live reload)

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Build the static site

```bash
npm run build
```

The static output is generated in the `out/` folder. You can preview it locally with:

```bash
npx serve out
```

Then open [http://localhost:3000](http://localhost:3000) (or the port shown in the terminal).

## Project Structure

```
app/
  layout.tsx   – HTML shell, metadata, fonts
  page.tsx     – Full one-page website (Nav, Hero, Services, About, Contact, Footer)
  globals.css  – Global styles and brand colors
public/
  logo.png     – J.Huser IT-Services logo
out/           – Static export (generated by `npm run build`)
```

## Deployment

The `out/` folder contains plain HTML/CSS/JS – upload it to any static host (Hostinger, Netlify, GitHub Pages, etc.).

Configure `next.config.ts` if you need a custom base path.
